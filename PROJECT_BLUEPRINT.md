## 一、百度股市通功能拆解

### 1. K线分析模块（用户发来的那张图）
| 功能 | 说明 |
|------|------|
| 时间周期 | 分时 / 日K / 周K / 月K |
| 复权方式 | 不复权 / 前复权 / 后复权 |
| MA均线 | MA5 / MA10 / MA20 / MA30 / MA60（可选颜色） |
| 技术指标 | MACD / KDJ / BOLL / VOL成交量 |
| 副图 | 成交量柱状图（VOL） |
| 操作 | 缩放 / 拖拽 / 十字光标 |

### 2. 智能选股（6套模板）
| 模板 | 筛选维度 |
|------|---------|
| 财务类 | 净利润增长率、营收增长率、毛利率、ROE |
| 技术类 | 均线多头、突破新高、量价齐升、MACD金叉 |
| 估值类 | PE<10、PE历史分位、PB<3、股息率>3% |
| 资金流 | 主力净流入、北向资金增持、换手率 |
| 趋势类 | 20日涨幅、60日涨幅、均线多头排列 |
| 风险类 | ST股、问题股、退市预警、商誉减值 |

### 3. 板块监控
- 行业板块涨跌幅排行
- 板块内资金净流入
- 板块龙头股
- 板块切换联动（点板块看成分股）

### 4. 资金流向
- 主力净流入/净流出
- 超大单/大单/中单/小单
- 北向资金（沪深港通）
- 南向资金（港股通）

### 5. 估值/财务/技术分析
- 估值分析：PE/PB/PCF历史分位
- 财务指标：营收/净利润/资产负债率
- 技术指标：MACD/KDJ/RSI/BOLL

### 6. 风险追踪
- 问题股筛选
- ST/\*ST预警
- 商誉减值预警
- 业绩变脸

## 五、技术方案

### 前端
- 框架：Vue3 + TypeScript + ECharts5
- K线图：echarts-for-react 或原生echarts
- UI风格：参考百度深色主题（#1a1a2e背景）

### 后端
- 框架：FastAPI
- MA计算：在Python层预计算，存入DB或实时计算
- 资金流：AKShare `stock_moneyflow` 系列接口

### 数据库
- 复用现有SQLite（stock_data.db）
- 新增 `moneyflow` 表
- 新增 `technical_indicators` 表（预计算MA/MACD/KDJ）
