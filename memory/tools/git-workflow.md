## 概述
Git 工作流配置，包括 /commit、/branch、/commit-push-pr 命令用法。

## 用法
```
/commit [--type=<type>] [--scope=<scope>] [--amend] [--dry-run]
/branch [list|create|switch|delete|status] [args]
/commit-push-pr [--message=<msg>] [--title=<title>] [--body=<body>] [--base=<branch>] [--dry-run] [--yes]
```

## 自动检测
- 改动检测
- AI 生成 conventional commit messages
- 分支管理
- PR 创建
