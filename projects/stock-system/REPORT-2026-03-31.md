### 1. 数据库 cron 修复 ✅

**问题：** `kline_component/kline_data.py` 读取空表 `quotes`，K 线数据返回为空

**根因：** 两套 API 走不同的数据源：
- `src/api/main.py`（8000端口）→ 正确读取 `daily_quotes`
- `kline_component/api_server.py` → 读取空的 `quotes` 表

**修复：** `kline_component/kline_data.py` 将两个查询方法从 `quotes` 切换到 `daily_quotes`，适配列名

**验证结果：**
```

✅ northbound_flow → 2026-03-30

```
